Overwatch Themed Alert Code for Twitch Alerts

To set up the your custom Twitch Alerts, follow these instruction:

  1. Log in to Twitch Alerts account
  2. Click the alert box widget tab
  3. Select the tab for the alert you’d like to set up
  4. Copy the appropriate message template code
  5. Replace the code in Twitch Alerts with the copied code
  6. Set Text Animation to none
  7. Remove any image inside of Twitch Alerts
  8. Set up your alert sound as normal
  9. Set alert duration to at least 8 seconds
  10. Important update (not in video): Set Alert Text Delay to 1 second in the general options.
  11. Under font settings, choose a text and highlight color
  12. Repeat this process for any other alerts you’d like to use this custom CSS for
  13. Click Save settings
  14. Under “How To” Check the boxes of the alerts you’d like to add in, and copy the URL
  15. In OBS Studio add a new Browser source, paste in the URL, set width to 1920 and height to 1080
  16. Copy the CSS code from this page
  17. Replace all CSS inside of OBS Studio with the CSS from this page
  18. Hit Ok and use the test buttons to verify that everything is working

Color Picker

To change the color of the background box, you will need to change the color code in the message template and replace the RGBA value. For example, to change it from blue with 80% opacity to a red with 30% opacity it would change from:

    style="background: rgba(37, 48, 74, .8)"

to

    style="background: rgba(211, 35, 0, .3)"
